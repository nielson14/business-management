For Teachers

Everything you need to know to use these simulations in your classroom — no setup required beyond sharing the link.

⚡
Zero Setup
Every simulation runs directly in the browser. No accounts, no logins, no software to install. Share the link and students start immediately.
🤖
AI Feedback Built In
Every simulation includes AI feedback that references each student's specific decisions by name — not generic encouragement. Students can't copy each other's feedback.
🏆
Escape Room Leaderboard
The escape room tracks class completion times in the browser. Use the "Clear Leaderboard" button between class periods. Students can compete across the day.
📝
Print to PDF
Every simulation has a Print / Save PDF button at the end. Students can submit their AI feedback report as a graded artifact directly from the browser.
🔀
Randomized Each Play
The Ethics simulator shuffles action orders each game. The escape room answer positions vary. Students can't share answers — they have to know the material.
📚
Recommended Sequence
Start with the Lemonade Stand (low stakes, fun). Build to Business Plan Builder and Financial Statements. Use the Escape Room as a final review before the cert exam.
